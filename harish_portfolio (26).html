Final Year Project · 2025–2026

Vision-Based Automatic Number Plate Recognition System

Designed and built a complete computer vision pipeline that detects and reads vehicle number plates from real-world images. The system works without manual intervention — it takes a raw photograph as input and outputs the plate number as text, handling variations in lighting, angle, and plate fonts.

The Problem

Manual vehicle identification is slow and error-prone. Traffic management, parking systems, and security checkpoints need fast, automated plate reading that works reliably across different conditions.

The Solution

Built an image processing pipeline: capture → grayscale conversion → noise reduction → edge detection → plate region localization → character segmentation → OCR text extraction. Each stage feeds the next for high accuracy output.

Key Challenges

Handling variable lighting conditions, skewed plates, and similar-looking characters (0/O, 1/I). Tuned preprocessing parameters to maximize detection accuracy across a diverse image dataset.

Outcome

Successfully recognized number plates from static images. The pipeline demonstrated consistent accuracy and forms a foundation for real-time vehicle tracking or parking automation systems.
